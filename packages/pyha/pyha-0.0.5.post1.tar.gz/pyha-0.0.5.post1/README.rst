=======
Install
=======

On Ubuntu 16.04, GHDL dependencies:

```
sudo apt install libgnat-4.9
```

```
pip install pyha
```

For running VHDL simulations, GHDL is needed. On Ubuntu it can be installed with

```sudo ./scripts/install_ghdl.sh```
