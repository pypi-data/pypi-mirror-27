detach-vnic
***********


Description
===========

Detaches and deletes the specified secondary VNIC. This operation
cannot be used on the instance's primary VNIC. When you terminate an
instance, all attached VNICs (primary and secondary) are automatically
detached and deleted.


Usage
=====

   oci compute instance detach-vnic [OPTIONS]


Options
=======


--vnic-id [text]
----------------

The OCID of the VNIC. [required]


--compartment-id, -c [text]
---------------------------

The OCID of the instance's compartment. [required]


--force
-------

Perform deletion without prompting for confirmation.


--from-json [text]
------------------

Provide input to this command as a JSON document from a file.

Options can still be provided on the command line. If an option exists
in both the JSON document and the command line then the command line
specified value will be used


-?, -h, --help
--------------

Show this message and exit.
