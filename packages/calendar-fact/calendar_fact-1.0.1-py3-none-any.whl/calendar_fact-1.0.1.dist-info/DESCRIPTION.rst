Calendar fact
=============

Taken from `xkcd.com/1930 <https://xkcd.com/1930/>`__, this command
generates calendar facts. Useful for MODs, head or foot lines in apps
etc.

Installation
------------

Install by running:

.. code:: shell

    pip install calendar-fact

Usage
-----

Run:

.. code:: shell

    calendar-fact
