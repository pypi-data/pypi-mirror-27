mds-account-invoice-deliverydate
================================
This module adds a delivery date to the invoice, which is necessary in Germany.
Default for invoicedate is set to 'today'.

Install
=======

pip install mds-account-invoice-deliverydate

Requires
========
- Tryton 4.6
