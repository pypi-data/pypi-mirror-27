screwdriver
***********

Another tool for your python toolset.  General odds and ends that kept coming
up in projects.

Installation
============

.. code-block:: bash

    $ pip install screwdriver

Supports
========

screwdriver has been tested with Python 2.7, 3.5, and 3.6
